# 功能介绍

## 多模型接入

一个工具统一接入主流模型服务，底层协议自动转换。

| 功能 | 说明 |
|------|------|
| 10+ Provider | 支持 GLM、DeepSeek、OpenAI、MiniMax、Mimo、Kimi、千问等 |
| 自动格式识别 | 主流模型协议自动适配，无需手动配置 |
| 模型切换 | 使用 `/model` 命令随时切换 |
| Provider 管理 | `/providers` 面板配置 API 密钥 |

## 超长期记忆

跨会话知识沉淀，支持全文搜索与语义搜索双路召回。

| 功能 | 说明 |
|------|------|
| 记忆持久化 | 分片式存储，自动蒸馏关键知识 |
| 全文搜索 | SQLite FTS5 全文索引 |
| 语义搜索 | Embedding 向量相似度匹配 |
| 上下文折叠 | 长对话自动压缩，不受上下文窗口限制 |

## Agent 与 Skill

| 功能 | 说明 |
|------|------|
| Agent | 定义角色和能力，支持并行执行 |
| Skill | 封装可复用流程，斜杠命令调用 |
| 模型偏好 | 每个 Agent 可指定不同模型 |
| 知识隔离 | scope 级别的知识管理 |

## 桌面自动化

| 功能 | 说明 |
|------|------|
| 窗口管理 | 绑定、激活、移动、缩放窗口 |
| 鼠标键盘 | 点击、滚动、输入、快捷键 |
| 截图分析 | 桌面窗口截图与 AI 分析 |
| 进程管理 | 启动和关闭桌面程序 |

## 开源情报分析

| 功能 | 说明 |
|------|------|
| 网页搜索 | 多搜索引擎支持，域名过滤 |
| 网页阅读 | URL 抓取并转为大模型友好输入 |
| 数据可视化 | 图表识别、趋势提取、异常检测 |
| 视频分析 | 场景理解、关键动作提取 |
| 技术图表 | 架构图、流程图、UML 解析 |

## 定时调度

| 功能 | 说明 |
|------|------|
| `/loop` | 循环计划任务 |
| Cron 调度 | 标准 5 字段 cron 表达式 |
| 持久化 | 任务重启后自动恢复 |
| 内存管理 | 长时间运行内存控制 |

## 工具系统

内置 56+ Agent 工具和 120+ 斜杠命令，覆盖文件操作、代码搜索、项目管理等场景。
